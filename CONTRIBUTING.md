Before submitting code changes, please open an Issue to discuss the proposed change. This helps ensure that contributions align with the project's roadmap and avoids duplicate work.

Please note that not every contribution will be accepted. All changes are reviewed to ensure they fit the project's goals and long-term direction.

Please be respectful and constructive in all discussions and contributions.
